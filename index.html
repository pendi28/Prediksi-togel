🧮 Rumus Prediksi Manual

Masukkan angka keluaran terakhir untuk menghitung berbagai rumus prediksi.

📖 Panduan Rumus

Rumus Ekor 2D
Kepala + Ekor kemarin = Result. Ambil satuannya.
Rumus Kepala 2D
(AS + KOP + Kepala + Ekor) % 10 = Kepala prediksi
Rumus Jitu
Ekor hari ini × 2 + 1 = Kandidat ekor besok
Rumus Lawan
Ekor + 5 (mod 10) = Angka lawan/pasangan
Rumus Kembang Kempis
Jumlah 4 digit → jika ganjil = kembang (+1 setiap digit), jika genap = kempis (-1 setiap digit)
Rumus Atas Bawah
Ekor 0-4 = Bawah, Ekor 5-9 = Atas. Biasanya bergantian.
